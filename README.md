# アルバイト給料見込み計算アプリ

テキスト形式のシフト表を貼り付けるだけで、アルバイトの月給見込み額を自動計算するStreamlitアプリです。

## 必要なライブラリのインストール

```bash
pip install streamlit jpholiday
```

## 実行方法

```bash
streamlit run app.py
```

ブラウザが自動で開き、アプリが表示されます。

## 使い方

1. シフト表をテキストエリアに貼り付けます
2. 「計算する」ボタンをクリック
3. 日別の内訳と合計が表示されます

### 対応フォーマット

```
日付	勤務時間	労働時間	休憩時間
12/18(木)	17:00～20:00	03:00	00:00
12/22(月)	17:00～20:00	03:00	00:00
01/03(土)	13:00～17:30	04:30	00:00
```

- 「－」「ー」を含む行は勤務なしとして無視されます
- 休憩時間は計算に含まれません

## 賃金ルール

| 条件 | 時給 |
|------|------|
| 基本時給 | 1,140円 |
| 土日祝日 | 1,290円 |
| 平日 13:00〜17:00 | 1,190円 |
| 平日 17:00以降 | 1,290円 |

- 時間帯をまたぐ場合は分割計算されます
- 祝日は `jpholiday` ライブラリで自動判定されます
- 深夜手当は考慮されません

## スクリーンショット

アプリを起動すると以下のような画面が表示されます：

- タイトル「アルバイト給料見込み計算」
- シフト表貼り付け用テキストエリア
- 「計算する」ボタン
- 結果表示エリア（日別内訳＋合計）
